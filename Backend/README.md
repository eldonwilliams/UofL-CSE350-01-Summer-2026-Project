# UofL-CSE350-01-Summer-2026-Project
Project for Intro to Software Engineering

Description of the project:

A real-time, collaborative virtual whiteboard web application that enables multiple users to brainstorm, sketch, and design simultaneously on a shared digital canvas. Built to enhance remote teamwork through seamless, instant synchronization and an intuitive user interface.

For the front end we will be using Java Script and for the backend we are going to use Java
